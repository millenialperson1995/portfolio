Pasta `projects/`

Coloque aqui páginas individuais dos seus trabalhos. Cada projeto pode ser:

- Uma página estática com descrição, imagens e links.
- Um demo hospedado localmente (ex: `projects/demo-landing.html`).

Boas práticas:
- Usar imagens em `../assets/img/` (caminho relativo quando a página estiver em `projects/`).
- Incluir metadados e um botão de retorno para a página principal (`../index.html`).
